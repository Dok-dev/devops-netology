# Monitoring

```
$ helm repo add prometheus-community https://prometheus-community.github.io/helm-charts
$ helm repo update
$ kubectl create namespace monitoring

$ helm -n monitoring install --replace -f ./configs/k8s/kube-prometheus-stack-values.yaml kube-prometheus prometheus-community/kube-prometheus-stack
$ helm -n monitoring upgrade -f ./configs/k8s/kube-prometheus-stack-values.yaml kube-prometheus prometheus-community/kube-prometheus-stack

$ kubectl get po,svc,ing -n monitoring 

$ kubectl --namespace monitoring port-forward svc/kube-prometheus-grafana 3000:80

$ kubectl apply -f grafana-ingress.yml -n monitoring
```
